# VouchID

Identity infrastructure for AI agents.

Every agent gets a cryptographic identity. Every MCP tool call is verified before it runs.

## Packages

- [`@vouchid/sdk`](./sdk) — Register agents, manage tokens, verify identities
- [`@vouchid/mcp`](./mcp) — Drop-in middleware for MCP servers

## Quick Start

```bash
npm install @vouchid/sdk @vouchid/mcp
```

See the individual package READMEs for full documentation.
